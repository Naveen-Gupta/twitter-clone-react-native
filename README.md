# twitter-clone-react-native
A Twitter clone using graphql and react-native.
# Steps:
~$ git clone
# For Server
server~$ npm install

# Add constants.js in server/src/config/



# For Mobile
mobile~$ npm install

# Setup for Expo to run app on Expo
Install Expo as global dependency
~$ npm install exp --global

# To Start with expo
mobile$ exp start
